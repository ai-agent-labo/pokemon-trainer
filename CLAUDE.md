# pokemon-trainer — Claude セッション handoff

ポケモンのタイプ相性を体で覚える学習ミニゲーム。Champions DB の補完アプリ。
リポ: https://github.com/ai-agent-labo/pokemon-trainer / Vercel デプロイ済 (静的サイト)。

## プロジェクト基本

- **構成**: 単一の `index.html` (HTML+CSS+JS 全部入り、73KB)
- **依存**: なし。ビルド不要
- **ローカル開発**: `python3 -m http.server 8766` → http://localhost:8766/
- **デプロイ**: GitHub に push すれば Vercel が自動で本番デプロイ

## ゲーム仕様

### モード

- **単タイプ** (`MONS_MONO`, 18 体): 1 タイプ 1 ポケモン、1 STAB 技
- **複合タイプ** (`MONS_DUAL`, 27 体): 2 タイプ、2 STAB 技
- 切替: ヘッダー右上「🎯 単タイプ / 🌀 複合タイプ」ボタン
- ベスト記録はモード別 localStorage (`*-mono` / `*-dual`)

### 複合プールの選定基準 (重要)

- Champions DB プールに存在すること (使用率は無関係、200位でも可)
- メガ進化でタイプが変わる mon は除外: **リザードン / ギャラドス / ムクホーク**
- 全 18 タイプを最低 2 回ずつ出現させる
- 「1 タイプ組み合わせ = 1 ポケモン」で記憶できるよう、組み合わせは全部ユニーク

### 難易度 (`levelFor` で試合数→Lv)

| Lv | 試合範囲 | 内容 |
|---|---|---|
| 1 観察 | 1〜15 | 全公開、相手交代なし |
| 2 読み合い | 16〜30 | 控え順ランダム、相手 1 回退避 (致命傷時 50%) |
| 3 リード読み | 31〜50 | Lv2 + 相手リードも非公開 (バトル開始まで) |
| 4 大荒れ | 51+ | Lv3 + 相手退避 2 回 + HP 半分以下でも 30% |

### スコア (`SCORE` 定数)

```js
撃破:    ×4 +150 / ×2 +100 / ×1 +40 / ×0.5 +20 / ×0.25 +10
倒された: 同上の負値
無効空振り (自分の技が無効): -50
無効ガード (相手の技を無効): +30
交代税: -5 (1回ごと)
試合勝利: +200 / 敗北: -100
完全勝利 (誰も落とされず): +150
ソロ突破 (交代せず勝利): +300
連勝倍率: 試合スコア × (1 + 連勝N × 0.1)
```

### 演出ルール

- **抜群 (×2)**: カードシェイク heavy + 画面シェイク + 赤フラッシュ + バナー黄枠
- **ちょうばつぐん (×4)**: カードシェイク **super** + 画面シェイク **super** + レインボーフラッシュ + バナーレインボーグラデ枠 + 爆発 SE (`hit_mega`)
- **無効 (×0)**: light シェイク + グレーフラッシュ
- **半減 (×0.5)**: light シェイク

### 相手 AI

- **退避**: 致命傷予感 (HP ≤ dmg) で 50%、Lv4 は HP 50 以下予感 30%
- **引き先**: 「最強の控え」ではなく **ランダム** (読み合い性のため)
- **退避ターンは反撃しない** (本家 Pokemon 仕様: 交代は priority +6 で先制)

### バリデータ (`validPuzzle`) — 正解保証

毎試合の盤面生成時に必ず満たす:

1. 相手の各タイプに対し、こちら最低 1 体が ≥×2 で殴れる
2. こちら 1 体で相手全員を ≥×2 ではない (交代必須)
3. こちら各 mon が、相手の少なくとも 1 体には ≤×1 で耐える (死に駒なし)
4. 相手 1 体がこちら全員に ≥×2 ではない (即詰みなし)

→ どのレベル・モードでも「正解が必ず存在する」puzzle のみ採用。

## 設計の歴史 (なぜ今こうなったか)

- **Lv2/Lv3 当初は情報非公開** だったが、バリデータが正解保証してるので「受け回すだけ」になり詰まらん → 廃止して「相手の交代」を Lv2 の本質に
- **相手の交代は最適マッチアップではなくランダム** (最適にすると 2 回目から読み切れて読み合いにならない)
- **ちょうばつぐん** は ひらがな表記 (「超抜群」じゃない)、レインボー専用演出
- **モード切替時はラン終了** (中途半端な状態を残さない)
